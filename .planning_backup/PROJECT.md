# 智慧旅居生成器 (AI Travelogue)

為解決多數人擁有海量旅行照片卻「懶得整理」的痛點，本專案旨在提供一個**低門檻、高質感產出**的網頁應用。使用者只需上傳旅行相片，系統即會透過 AI 進行相片優化與篩選，自動產生帶有優雅動畫且賞心悅目的「網頁版遊記」，並可一鍵分享給親朋好友。

## Core Value
解決「低門檻製作」與「高質感產出」的矛盾，將數百張未整理的相片，一鍵轉換為極具吸引力且有動態視覺體驗的精美回憶錄。
