# Roadmap

## Phase 1: Foundation & Web Shell
**Goal:** 初始化基礎 Next.js/Vite 架構，設定 TailwindCSS 與路由系統，並建立專案底層 UI 框架。

## Phase 2: Photo Upload & Storage System
**Goal:** 實作前端的相片大量上傳元件，並將圖片保存至安全的雲端存儲 (e.g., Supabase storage/AWS S3)。

## Phase 3: AI Curation Engine (相片智選引擎)
**Goal:** 建立後端或 Serverless API，整合 AI 將上傳的圖片組進行評分、去重、優化，並篩選出最佳相片集合。

## Phase 4: Dynamic Travelogue UI 
**Goal:** 開發高質感、具備捲動動畫(Scroll Animations)的遊記展示頁面，負責將 Phase 3 篩選出的結果精美呈現。

## Phase 5: Sharing Flow & Final Polish
**Goal:** 實作獨一無二的分享連結生成系統，確保其他人能在手機上流暢觀看，完成視覺打磨。
